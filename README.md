# Beaverdale Little League Website

A modern, responsive website for Beaverdale Little League.

## Features

- Responsive design that works on desktop, tablet, and mobile devices
- Smooth scrolling navigation
- Clean, professional layout
- Easy to customize and maintain

## Project Structure

```
BeaverdaleLittleLeague/
├── index.html          # Main HTML file
├── styles.css          # Stylesheet
├── script.js           # JavaScript functionality
├── .gitignore          # Git ignore file
└── README.md           # This file
```

## Getting Started

Simply open `index.html` in your web browser to view the website locally.

## Deployment

This is a static HTML website and can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Any web hosting service

## Customization

- Edit `index.html` to change content and structure
- Modify `styles.css` to adjust colors, fonts, and layout
- Update `script.js` to add or modify interactive features
